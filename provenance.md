# AI Provenance

This browser extension, **IP Address & Time**, was developed with the assistance of artificial intelligence.

## AI Involvement

- **Code Generation**: The majority of the HTML, CSS, and JavaScript logic was generated or refined using AI models (specifically Antigravity, powered by Google's Gemini).
- **Design & Styling**: The visual design, including the dark mode aesthetic and responsive layout, was designed with AI guidance.
- **Refactoring & Debugging**: AI was used to identify and fix bugs, such as CSS contrast issues and code duplication.

## Human Oversight

While AI provided the foundational code and design, human developers provided the creative direction, reviewed all generated code for security and performance, and made final decisions on features and functionality.
