# UserAgent-MiscHeaders

A Firefox extension for managing User-Agent and other miscellaneous HTTP headers.

## Features

- Modify User-Agent header.
- Manage custom HTTP headers.
- Firefox compatible.

## Installation

1. Clone the repository.
2. Open Firefox and go to `about:debugging`.
3. Click "This Firefox".
4. Click "Load Temporary Add-on" and select the `manifest.json` file.

## Add to Firefox

This extension is already published on the [Mozilla Add-ons website](https://addons.mozilla.org/en-US/firefox/addon/useragent-misc-headers/).
